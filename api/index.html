Corona Case Information Service

Welcome to this restful API. The API consists of the following endpoints:

  • /corona/v1/country/
  • /corona/v1/policy/
  • /corona/v1/diag/
  • /corona/v1/notifications/
  • /corona/v1/viewhooks/

The initial endpoint focuses on returning the number of confirmed and recovered Covid-19 cases for a given country.
Optionally, the user can specify a date range.

GET Path: /corona/v1/country/{:country_name}{?scope=begin_date-end_date}



The second endpoint provides an overview of the current stringency level of policies regarding Covid-19 for given countries.

GET Path: /corona/v1/policy/{:country_name}{?scope=begin_date-end_date}



The diagnostics interface indicates the availability of all individual services this service depends on.
It wil also inform you about the version, uptime and number of webhooks registered on the service

GET Path: /corona/v1/diag/



The notifications endpoint allow you to view, register new and delete webhooks from the firestore database.
You can click here to see the activity.

GET/POST/DELETE Path: /corona/v1/notifications/



The last endpoint allows you to see all registered webhooks in the browser, for simplicity's sake.

GET/POST/DELETE Path: /corona/v1/viewhooks/
