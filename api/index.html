CryptoNotify

Welcome to CryptoNotify. This service allows you to get real-time notifications on cryptocurrencies directly to your cellphone.

CryptoNotify offers the following services:

  • Real-time notifications on price-targets
  • Real-time notifications on trends
  • Real-time notifications on changes in your own portfolio
  • Real-time prices on the 100 most popular currencies on this website

Use the following endpoint to view real-time prices of the 100 most popular currencies:

GET Path: /cryptonotify/v1/currency/

This endpoint can be very useful when registering notifications as you will need the currency's symbol:



You can also focus on a single currency using its name or symbol as input:

GET Path: /cryptonotify/v1/currency/{:symbol/name}



Here is an example of how it works

GET Path: /cryptonotify/v1/currency/Bitcoin GET Path: /cryptonotify/v1/currency/btc



In order to register for our notification-service. Use Postman to perform a post-request.
This is how to sign up for Trend notifications:

POST Path: /cryptonotify/v1/trends

And fill in your information and trend-target in the following format:

{
"url": "webhook.site/something/something",
"phone_number": "+4795833037",
"symbol": "ETH",
"percentage_threshold": 3
}


Where the percentage_threshold represents the percentage change in volume for the selected currency.
Negative values for the percentage threshold also works in case you want notifications about volume reduction.
"symbol" represents the symbol of the chosen currency. Use the similar format like btc for bitcoin or ada for cardano.
You WILL need to provide either a phone number or a url.





If you want to sign up for a price-target notification the same rules apply:

POST Path: /cryptonotify/v1/pricetarget

And fill in your information and trend-target in the following format:

{
"url": "webhook.site/something/something",
"phone_number": "+4797885707",
"symbol": "XRP",
"target_price": 2
}


Where the target_price represents the price in US dollars. The notification will be sent out as soon as the currency's price hits it.
"symbol" represents the symbol of the chosen currency. Use the similar format like btc for bitcoin or ada for cardano.
You WILL need to provide either a phone number or a url.
